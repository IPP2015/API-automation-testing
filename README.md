API Automation Testing

Ini adalah project API Automation Testing menggunakan Katalon Studio

Folder dan isinya
Reports - hasil test dan log
Test Suites - kumpulan test suite
Test Cases - test case individual


Cara Pakai

1. Pastikan Katalon Studio sudah terpasang di komputer
2. Clone project dari GitHub
3. 
4. Buka project di Katalon Studio
   File - Open Project - pilih folder project
5. Jalankan test suite atau test case sesuai kebutuhan
6. Semua hasil akan muncul di folder Reports
   -kalo free hanya ada di folder tidak bisa langsung di katalon studio
   -kalo test run test case tidak dapat report kalo test run test suite terdapat report
   -ada report pertama saya dengan nama folder 20260324_190940

Catatan Penting

Gunakan profil environment yang sesuai
Semua fungsi tambahan ada di folder Libs
Pastikan semua plugin atau library Katalon yang dibutuhkan sudah terpasang
